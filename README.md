# E-Com Backend API

A backend REST API for an e-commerce application built using Node.js, Express.js, and MongoDB.
This project focuses on clean API structure, database integration, and scalable backend logic.

## Features
- RESTful API architecture
- Product and user related CRUD operations
- MongoDB database integration using Mongoose
- Express.js routing and controllers
- Modular folder structure
- Error handling and clean code practices

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- JavaScript (ES6+)

## Project Structure
- routes/       → API routes
- controllers/ → Business logic
- models/      → Database schemas
- config/      → DB configuration
- server.js    → Entry point

## How to Run Locally

1. Clone the repository
   git clone https://github.com/Ajay110125yadav/E-Com

2. Install dependencies
   npm install

3. Setup environment variables
   - Add MongoDB connection string

4. Start the server
   npm start

## Purpose of This Project
This project was built to practice backend development concepts such as API design,
database operations, and scalable server-side logic.
